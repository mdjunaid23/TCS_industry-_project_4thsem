# 📊 Retail Sales Performance Dashboard

## 🚀 Project Overview

This project presents an **interactive Retail Sales Performance Dashboard** built using Power BI. It analyzes retail transaction data to provide insights into sales trends, customer behavior, and product performance.

The dashboard helps businesses make **data-driven decisions** by visualizing key metrics and patterns in an easy-to-understand format.

---

## 🎯 Objectives

* Analyze retail sales data
* Identify trends and patterns
* Understand customer demographics
* Evaluate product category performance
* Build an interactive dashboard

---

## 📂 Dataset

The dataset contains the following fields:

* Transaction ID
* Date
* Customer ID
* Gender
* Age
* Product Category
* Quantity
* Price per Unit
* Total Amount

---

## 🛠️ Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Excel (optional preprocessing)

---

## 📊 Dashboard Features

* 📌 KPI Cards (Total Sales, Quantity, Transactions, Average Sales)
* 📈 Sales Trend Analysis (Line Chart)
* 📊 Product Category Performance (Bar Chart)
* 🥧 Gender Distribution (Pie Chart)
* 🍩 Age Group Analysis (Donut Chart)
* 🎛️ Interactive Filters (Slicers)

## 🔍 Key Insights

* Sales show a declining trend over time
* Electronics and Clothing are top-performing categories
* Beauty category has lower contribution
* Customer distribution is balanced across gender
* Age group 36–50 contributes the highest sales

---

## 💡 Business Recommendations

* Target younger customers (18–25) with marketing campaigns
* Promote underperforming categories (Beauty)
* Offer discounts during low-performing periods
* Focus inventory on high-demand products

---

## ⚠️ Limitations

* Dataset is not real-time
* External market factors are not included
* No region-wise analysis available

---

## 🔮 Future Scope

* Real-time data integration
* Predictive analytics (sales forecasting)
* Customer segmentation using machine learning
* Advanced business intelligence features

---

## 📌 Conclusion

This project demonstrates how data visualization and analytics can improve business decision-making. The Power BI dashboard provides clear insights into sales performance and customer behavior, making it a valuable tool for retail analysis.
